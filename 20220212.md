# Belajar PHP Loop

## While Loop

```php

<?php
  $i = 0;
  while($i < 5) {
    echo $i;
  }
?>
```

## For Loop

```php

<?php
  for($i = 0; $i < 5; $i++) {
    echo $i;
  }
?>
```

## Foreach Loop

```php

<?php
  $myarray = [0, 1, 2, 3, 4];
  foreach($myarray as $item) {
    echo $item;
  }
?>
```
